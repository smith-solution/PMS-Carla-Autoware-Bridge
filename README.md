## Introduction
CARLA-Autoware-Bridge는 CARLA-ROS-Bridge의 도움으로 CARLA 시뮬레이터를 Autoware Core/Universe에 연결하는 패키지입니다. 현재 최신 Autoware Core/Universe와 CARLA 0.9.15를 지원합니다.
